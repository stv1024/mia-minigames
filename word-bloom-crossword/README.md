# Mia's Word Bloom

一个给小学低年级使用的固定关卡英文 crossword 拼词小游戏试样。

## 玩法

- 直接打开 `index.html`。
- 点击格子或线索后，用键盘输入字母。
- 每 5 关为 1 章，完成章节后开启宝箱并获得固定徽章。
- 徽章和闯关进度保存在浏览器 `localStorage`。

## 当前设计

- 技术栈：Vue 3 ESM CDN + 原生 HTML/CSS。
- 关卡：50 个固定关卡蓝图，写在 `LEVEL_WORDS`。
- 词库：来自提供的单词列表，释义和图标写在 `WORDS`。
- 难度曲线：前 10 关 3 个短词，之后逐步增加长词、复习混合词和 5-6 词关卡。
- 徽章：10 个固定章节徽章，写在 `BADGES`，不随机、不重复。

后续如果要精修关卡，优先调整 `LEVEL_WORDS` 的分组顺序；如果要换提示或图标，调整 `WORDS`。
