# 弹珠台 Marble Board

一款融合 Pachinko（弹珠机）与 Plinko（钉板游戏）的 2D 休闲小游戏。

## 玩法

1. **按住**右下角发射器蓄力（按得越久，力量越大）
2. **松手**发射弹珠
3. 弹珠会撞击钉子和偏向板随机弹落
4. 底部亮灯的格子是目标，落入得分：
   - 亮灯中央格：**300 分**
   - 亮灯普通格：**150 分**
   - 未亮灯格：**0 分**
5. 共 **5 颗球**，每轮随机亮 3 个格子

## 技术栈

- HTML5 Canvas
- [Matter.js](https://brm.io/matter-js/) 2D 物理引擎

## 启动

直接用浏览器打开 `index.html` 即可，无需任何构建步骤。
